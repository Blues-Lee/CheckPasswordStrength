CheckPasswordStrength
==
CheckPasswordStrength是一个Java开源库，提供检测密码强度的功能。

Usage
--
```Java
    public static void main(String[] args) {
		String passwd = "2hAj5#mne-ix.86H";
		System.out.println(CheckStrength.checkPasswordStrength(passwd));
	}
```
